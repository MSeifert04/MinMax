Class MinMaxByExt

The MinMaxBy extension methods for System.Collections.Generic.IEnumerable<T>

The MinMaxBy methods return the minimum and maximum item of the enumerable by comparing the result of a key-function applied to the items. The functions will not return the result of the key-function but the actual item that had the minimum/maximum key. In this regard these methods differs from the overload of Min and Max in the System.Linq namespace that take a function to **transform** the elements.
